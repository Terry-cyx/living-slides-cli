DATABASE RATIONALE

为什么选择 PostgreSQL 作为交易主库

1. ACID + MVCC

原生多版本并发控制，长事务不阻塞读，金融级一致性保障。

2. JSONB 一等公民

订单元数据可同时享受关系建模与文档灵活性，避免双数据库。

3. 扩展生态

Citus 水平分片、TimescaleDB 时序、pgvector 向量检索一站到位。

4. 分区与并行查询

声明式分区 + 并行执行，支持单库轻松扩展到 10TB。

5. 开源治理稳定

PostgreSQL 全球社区，无单一厂商绑定，云厂商一致支持。
