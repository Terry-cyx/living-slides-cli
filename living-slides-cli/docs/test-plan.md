# htmlcli 集成测试计划

> 模拟真实 PPT 使用场景，测试 htmlcli 的完整 AI 协作工作流：
> AI 生成 HTML → 用户可视化编辑 → AI 读取 changelog → AI 润色 → 再次编辑 → 循环

## 测试方法

每个测试用例模拟 3 轮交互：
- **Round 1**: AI 根据需求生成 HTML 文件
- **Round 2**: 模拟用户可视化编辑（通过 API 修改 HTML），验证 changelog 生成
- **Round 3**: AI 读取 changelog 后进行智能润色，验证润色质量

## 评估标准

| 维度 | 通过条件 |
|------|---------|
| 生成质量 | HTML 包含所需的结构和内容 |
| 编辑追踪 | changelog 准确记录了所有变更 |
| 润色智能 | AI 基于 changelog 做出合理的改进，不破坏用户的修改 |
| 文件完整性 | 每轮交互后 HTML 文件可正常渲染 |

## 测试用例

### Test 01: 公司季度汇报 (QBR)
**需求**: 制作一个 Q1 季度业绩汇报，包含收入、用户增长、客户满意度等 KPI
- Round 1: 生成包含 4 个 KPI 卡片 + 趋势图区域 + 行动计划的 HTML
- Round 2: 用户修改收入数字从 ¥500 万改为 ¥620 万，增长率从 12% 改为 18%
- Round 3: AI 读取变更，自动调整相关文字描述（如"超出目标"），保持数据一致性

### Test 02: 产品发布会演示
**需求**: 新产品发布演示文稿，包含产品特性、定价、竞品对比
- Round 1: 生成封面页 + 产品特性页 + 定价页 + 竞品对比表 + 结束页
- Round 2: 用户修改产品名从"SmartBot"改为"AI助手"，修改定价从 ¥99 改为 ¥79
- Round 3: AI 读取变更，确保全文产品名统一，调整定价页的描述文字

### Test 03: 团队周会
**需求**: 简短的团队周会 slides，包含本周进展、下周计划、阻塞问题
- Round 1: 生成 3 页简洁的周会 HTML
- Round 2: 用户添加一个新的阻塞问题"服务器扩容审批延迟"
- Round 3: AI 读取变更，建议增加一个"需要协助"标签，优化排版

### Test 04: 培训课件 — Python 入门
**需求**: Python 编程入门课件，包含代码示例、练习题
- Round 1: 生成包含代码高亮的教学 slides（变量、循环、函数）
- Round 2: 用户修改代码示例，将 `print("Hello")` 改为 `print("你好世界")`
- Round 3: AI 读取变更，同步更新代码注释和说明文字为中文

### Test 05: 销售提案
**需求**: 给客户的销售提案，包含公司介绍、解决方案、成功案例、报价
- Round 1: 生成专业的销售提案 HTML
- Round 2: 用户修改客户名从"XX公司"改为"字节跳动"，修改报价金额
- Round 3: AI 读取变更，调整方案描述匹配客户行业特征

### Test 06: 项目进度报告
**需求**: 项目里程碑进度报告，包含甘特图样式的进度条、风险评估
- Round 1: 生成项目进度页面，包含进度条组件和里程碑列表
- Round 2: 用户将"后端开发"的进度从 60% 改为 85%，将状态从黄色改为绿色
- Round 3: AI 读取变更，更新整体项目完成度，调整风险评估

### Test 07: 数据分析报告
**需求**: 用户行为数据分析报告，包含漏斗图、转化率、用户画像
- Round 1: 生成数据分析页面，用 CSS 实现简单的图表效果
- Round 2: 用户修改转化率数据，将注册转化从 15% 改为 22%
- Round 3: AI 读取变更，调整分析结论和建议

### Test 08: 技术架构评审
**需求**: 系统架构评审 slides，包含架构图、技术选型对比、性能指标
- Round 1: 生成架构图 + 技术对比表 + 性能数据页
- Round 2: 用户在技术对比表中将数据库从 MySQL 改为 PostgreSQL
- Round 3: AI 读取变更，更新架构图中的数据库组件，调整技术选型理由

### Test 09: 投资人 Pitch Deck
**需求**: 给投资人的融资演示，包含市场规模、商业模式、团队、财务预测
- Round 1: 生成 8-10 页的 Pitch Deck
- Round 2: 用户修改融资额从 500 万改为 1000 万，修改估值
- Round 3: AI 读取变更，重新计算股权比例描述，调整资金用途分配

### Test 10: 年终总结
**需求**: 个人/部门年终总结，包含成果展示、数据亮点、明年规划
- Round 1: 生成年终总结 HTML，包含时间线、成果卡片、数据亮点
- Round 2: 用户添加一项新成果"完成 AI 客服系统上线"
- Round 3: AI 读取变更，调整成果数量统计，优化布局

### Test 11: 会议议程
**需求**: 正式会议的议程安排，包含时间表、议题、发言人
- Round 1: 生成会议议程页面，时间表格式
- Round 2: 用户调整某议题时间从 30 分钟改为 45 分钟，后续时间顺延
- Round 3: AI 读取变更，自动调整后续所有议题的时间

### Test 12: 竞品分析报告
**需求**: 竞品功能对比分析，包含多维度评分表、优劣势总结
- Round 1: 生成竞品对比矩阵 + 评分雷达图（CSS 实现）+ SWOT
- Round 2: 用户新增一个竞品"产品D"到对比表中
- Round 3: AI 读取变更，更新分析结论，调整排名
